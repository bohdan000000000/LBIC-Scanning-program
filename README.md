LBIC Scanning Program
Про проєкт
Низьковартісна автоматизована система керування портативною LBIC-установкою (Light Beam Induced Current) для локальної фотоелектричної діагностики фоторекомбінаційних сенсорів та сонячних елементів.
Метод LBIC дозволяє отримувати двовимірні карти рекомбінаційної активності та розраховувати внутрішню квантову ефективність із мікрометровою просторовою роздільною здатністю. Комерційні LBIC-системи базуються на дорогих прецизійних платформах та lock-in підсилювачах дослідницького класу. Даний проєкт реалізує аналогічний функціонал на базі доступних мікроконтролерів та вторинно використаної механіки CD-ROM приводів.

Архітектура системи
PC (Python)
  ├── GRBL (Arduino #1, COM3) — XY-рух через CD-ROM каретки
  └── Вимірювальний вузол (Arduino #2, COM4) — лазер + АЦП + FSM
На кожній точці сканування:

Arduino #2 вимірює темновий сигнал (лазер OFF) з oversampling ×64
Вмикає лазер, вимірює світловий сигнал (лазер ON)
Надсилає диференційний сигнал ΔV = V_light − V_dark на ПК

Алгоритми обробки сигналу
Передискретизація (Oversampling) — Arduino #2
64 відліки АЦП → сума >> 3 → ефективна роздільна здатність ~13 біт з 10-бітного АЦП.
Час стабілізації після перемикання лазера: 5000 мкс (SETTLE_US).
FSM вимірювального циклу
IDLE → (команда MEASURE)
  → DARK  : лазер OFF → oversampling → v_dark
  → LIGHT : лазер ON  → oversampling → v_light → лазер OFF
  → SEND  : Serial.println("DATA:" + (v_light − v_dark))
  → IDLE
Обробка на ПК

Корекція фону — віднімання середнього по перших і останніх 5 рядках (поза зразком)
Нормування — лінійне масштабування до діапазону [0, 1]
Гаусове згладжування — опціонально, σ = 0.5 за замовчуванням

Компенсація люфту CD-ROM (~0.3–0.5 мм)
Перед кожним рядком каретка відходить на OVERSHOOT = 0.5 мм по −X, потім рухається в напрямку +X — систематична похибка люфту скасовується.

Застосування
Установка апробована для:

Картографування локальної ділянки сонячного елемента — виявлення дефектних зон із зниженою квантовою ефективністю
Дослідження фоторекомбінаційного сенсора — реєстрація зміни фотоструму при дії парів ізопропілового спирту
